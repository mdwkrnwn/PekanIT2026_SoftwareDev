# 📌 Tentang Project

**UFinder** adalah website berbasis web modern untuk membantu pengguna menemukan dan mengelola informasi atau layanan tertentu dengan tampilan yang sederhana, responsif, dan mudah digunakan. Website ini menggunakan teknologi frontend modern serta di-host melalui platform Vercel untuk performa yang cepat dan ringan.

---

# 🚀 Instalasi & Menjalankan Project

## 1. Clone Repository

```bash
git clone https://github.com/mdwkrnwn/SEFEST26WEBDESIGN_LOW-CORTISOL.git
```

## 2. Masuk ke Folder Project

```bash
cd SEFEST26WEBDESIGN_LOW-CORTISOL
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Jalankan Development Server

```bash
npm run dev
```

Project akan berjalan di:

```bash
http://localhost:3000
```
