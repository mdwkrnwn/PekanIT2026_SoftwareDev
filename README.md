# 📌 Tentang Project

**Bakool** adalah website berbasis web modern untuk membantu pengguna menemukan dan mengelola informasi atau layanan tertentu dengan tampilan yang sederhana, responsif, dan mudah digunakan. Website ini menggunakan teknologi frontend modern serta di-host melalui platform Vercel untuk performa yang cepat dan ringan.

---

# 🚀 Instalasi & Menjalankan Project

## 1. Clone Repository

```bash
git clone https://github.com/mdwkrnwn/PekanIT2026_SoftwareDev.git
```

## 2. Masuk ke Folder Project

```bash
cd PekanIT2026_SoftwareDev
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Jalankan Development Server

```bash
npm run dev
```

Project akan berjalan di:

```bash
http://localhost:3000
```
